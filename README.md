Aplicativo mobile para gestão de cuidados de idosos, desenvolvido com foco em organização de rotinas, controle de medicações e acompanhamento de pacientes por cuidadores.

---

## 📱 Sobre o projeto

O projeto foi desenvolvido com o objetivo de auxiliar cuidadores de idosos no gerenciamento das atividades diárias, reduzindo erros e aumentando a qualidade do cuidado prestado.

A ideia surgiu a partir da análise de dificuldades reais enfrentadas por cuidadores, como:

* Controle manual de medicações
* Falta de organização das atividades
* Dificuldade de acompanhamento por familiares
* Ausência de histórico confiável

---

## 🚀 Funcionalidades atuais (MVP)

* 📋 Cadastro de pacientes
* 💊 Controle de medicações (visualização de pendentes)
* 📊 Resumo diário (dashboard)
* ✅ Registro de atividades
* 👥 Listagem de pacientes

## 📸 Demonstração

O aplicativo conta com uma interface simples e intuitiva, focada em facilitar o uso no dia a dia do cuidador.

Principais telas:

* Tela inicial (dashboard)

* Resumo de atividades e medicações
